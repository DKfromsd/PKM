## Add your files  

* [Create](https://docs.gitlab.com/ee/user/project/repository/web_editor.html#create-a-file) or [Upload](https://docs.gitlab.com/ee/user/project/repository/web_editor.html#upload-a-file) files
* [Add files using the command line](https://docs.gitlab.com/topics/git/add_files/#add-files-to-a-git-repository) or push an existing Git repo with the following comnad:

```
cd existing_repo
git remote add origin https://gitlab.com/yourcompany/team/dev/prj1/your-prj.git
git branch -M corp/main
git push -uf origin corp/main
```
